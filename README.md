# Task Manager App

A simple full-stack task manager application built with Node.js, Express, MySQL, HTML, CSS, and JavaScript.

## Live Demo

https://task-manager-app-production-a546.up.railway.app

## Features

- Create tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Set priority
- Add due dates
- Filter and sort tasks
- Pagination
- Responsive design

## Tech Stack

- Node.js
- Express
- MySQL
- HTML
- CSS
- JavaScript
- Railway

## Local Setup (if you don't want to use the Live Demo)

Clone the repository, install dependencies, create a `.env` file, run the SQL in `schema.sql`, and start the server.

```bash
git clone https://github.com/Ovione97/Task-Manager-App.git
cd Task-Manager-App
npm install
node server.js
